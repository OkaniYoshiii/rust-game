# Rust Game

## Concept

Vous gérez une entreprise qui créé de la nourriture pour animaux.
Le jeu est un genre de jeu d'automatisation à la manière de Satifactory mais en vue de haut.
La création de nourriture pour animaux peut engendre des conséquences sur le monde dans lequel vous fondez votre entreprise (chats errants, rats, augmentation du bonheur etc ...).
La ville peut, en fonction des actions que vous prenez, décider de couper le courant, vous taxer ou réaliser d'autres actions positives ou négatives à votre encontre.
La création de l'entreprise en son fonctionnement nécessite aussi de gérer les employés, leurs salaires, conditions de travail, pauses calins etc ...
Le jeu sera relativement simple comme l'est City Skylines par exemple.
L'augmentation du nombre d'employés pourra nécessiter l'achat de nouvelles aquisitions pour créer des parkings par exemple mais pourra créer des bouchons si vous ne le faites par ou avez une entreprise trop grosse par rapport à la localisation de cette dernière dans la ville.
Vous aurez plusieurs entreprises à gérer qui s'occuperont de chaque étape de création de vos produits. Il faudra donc gérer le trafic pour passer les marchandises entre entreprises pour rendre ça le plus efficace possible.
